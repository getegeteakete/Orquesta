# freee AI CFO チャットボットシステム

チャット形式でfreeeと連携し、**財務分析・自動記帳・請求書作成・異常検知**を行うAIアシスタントです。

## 主な機能

| 機能 | 概要 |
|------|------|
| 📊 財務分析 | 損益・BS・キャッシュフローをチャット+グラフで即座に確認 |
| 📝 自動記帳 | 自然言語の指示で仕訳・取引を自動登録 |
| 📄 請求書作成 | 取引先・金額を伝えるだけで請求書を下書き保存 |
| 🔍 異常検知 | 二重計上・外れ値・消費税区分ミスを自動検出 |

## セットアップ

### 1. freee デベロッパー登録

1. [freee デベロッパーポータル](https://developer.freee.co.jp/) にアクセス
2. アプリを作成し、以下を設定：
   - **リダイレクトURI**: `http://localhost:3000/api/freee/auth/callback`
   - **スコープ**: `read write`
3. Client ID と Client Secret を取得

### 2. 環境変数設定

```bash
cp .env.example .env.local
# .env.local を編集して各値を設定
```

### 3. インストール・起動

```bash
npm install
npm run dev
# http://localhost:3000 でアクセス
```

### 4. freee 認証

```
http://localhost:3000/api/freee/auth でOAuth2認証を開始
→ freee の認証ページにリダイレクト
→ 許可後、コールバックURLにリダイレクト
→ アクセストークンがCookieに保存
```

## 実装済みツール一覧

### 財務分析（5ツール）
- `get_profit_loss` - 損益計算書取得
- `get_balance_sheet` - 貸借対照表取得
- `get_monthly_trend` - 月次推移グラフ
- `simulate_cash_flow` - 資金ショートシミュレーション
- `get_expense_breakdown` - 販管費内訳（除外条件付き）

### 記帳・入力（3ツール）
- `create_expense_journal` - 経費仕訳登録（立替対応）
- `register_invoice_as_payable` - 請求書→買掛金登録
- `get_account_items` - 勘定科目一覧取得

### 請求書・見積書（3ツール）
- `create_invoice_draft` - 請求書下書き作成
- `get_renewal_clients` - 更新月クライアント一覧
- `get_partners` - 取引先検索

### 異常検知・監査（3ツール）
- `detect_anomalies` - 二重計上・外れ値検出
- `check_tax_codes` - 消費税区分チェック
- `get_deals_for_audit` - 監査用取引データ取得

## 書き込み操作の安全設計

```
ユーザー: 「タクシー代5,000円を登録して」
    ↓
Claude: ツール使用を決定（create_expense_journal）
    ↓
APIサーバー: 書き込み操作を検出 → 確認ダイアログを表示
    ↓
ユーザー: 「実行する」ボタンをクリック
    ↓
freee: 仕訳登録 → 完了メッセージ
```

書き込み操作（仕訳登録・請求書作成）は**必ず確認ダイアログを表示**してから実行します。

## 本番化に向けて必要な作業

| 項目 | 現状 | 本番対応 |
|------|------|---------|
| トークン管理 | Cookie保存（デモ） | DB + Redis でセッション管理 |
| マルチテナント | 未対応 | ユーザーごとのトークン分離 |
| レート制限 | なし | freee API制限（500req/h）対応 |
| エラーハンドリング | 基本実装 | Retry・フォールバック実装 |
| ログ監査 | なし | 誰がいつ何を登録したかの記録 |
| PDF/画像解析 | 未実装 | Claude Vision でBase64送信 |
